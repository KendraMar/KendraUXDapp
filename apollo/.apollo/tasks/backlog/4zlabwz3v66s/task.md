# New Canvas Node: URLs/Prototypes

## Description

Canvas documents should support a new node type of URL, which supports pulling in a static cached image of the URL and/or prototype and then optionally clicking it or opening it in a new tab to view. This would enable us to have Canvas-based journey maps that can show what steps a user would take next on the actual page itself, at the correct URL.
