# Recording AI chapter markers

## Description

Similar to manual and automatically-generated chapter markers in YouTube and Snipd, add support for AI-suggested markers that can be modified manually if needed. The model used to suggest the markers and when the markers were generated should be stored as metadata for tracking purposes so that better markers can be created. If a person adds manual chapter markers or makes modifications, those should be preserved as well.
