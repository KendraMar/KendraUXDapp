# Extended Displays — Design History

## 2026-02-10

### [Addition] Initial concept exploration for multi-window awareness
- Created spike to explore "Extended Displays" — the idea that multiple Apollo windows on the same machine should be aware of each other and coordinate their content. Brainstorming document covers coordinated navigation, distributed interface, context radiators, saved workstation layouts, AI-augmented display management, and knowledge explorer extensions. Technical approach favors BroadcastChannel API for basic awareness with WebSocket hub for server-side coordination.
