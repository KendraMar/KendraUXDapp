# Design History

## 2026-02-07

### [Addition] Sidebar navigation with spaces
- Sidebar provides primary navigation with space-switching support
- Supports pinned conversations, docked chats, and agent-suggested nav items
- State persisted to localStorage (open/collapsed/hidden)

### [Decision] Space-based navigation model
- Adopted a "spaces" concept to organize navigation around project areas
- Each space can have its own set of pinned items and layout
