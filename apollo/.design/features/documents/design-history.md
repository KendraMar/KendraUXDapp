# Documents - Design History

## History

### 2026-02-10

- **[Addition]** Added inline document discussions feature (Google Docs-style commenting). Users can select text in the rich editor, click a floating "Comment" button, and start a threaded discussion anchored to that text. A collapsible sidebar panel shows all active discussions with filtering by open/resolved status. Highlighted text in the editor indicates active discussions; clicking highlights opens the corresponding thread. Supports threaded replies and basic formatting (bold, italic, code) in comments.
